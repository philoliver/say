Hello there!

We want foobar to be your company. We want to hear your opinion about things. We want to talk!
But we also know that sometimes, it is difficult to say things up front. That's why we have created 'Say!'

Have something to say about the company, but afraid to let everyone know it was you? Worry not! 'Say!' helps you do just that. You can post things here anonymously! Don't let the things you think can be improved, remain unspoken, or even things you think are not going quite so well!

Code of cunduct

There is only one thing we are asking you when you use this site: Respect for everyone!

This is an anonymous site. We think it empowers everyone to speak freely. However, this freedom has to be used carefully. Please, think twice before you post something. When criticizing, don't use names. Please mind your language!
